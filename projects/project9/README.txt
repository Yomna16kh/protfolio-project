# Eventique | עיצוב אירועים באווירה אחרת

## תיאור
דף נחיתה מעוצב למעצבת אירועים, המשלב אסתטיקה גבוהה, גלריית תמונות עבודות, טופס יצירת קשר ואייקונים חברתיים. האתר עוצב בצבעי גרדיאנט חמים עם טיפוגרפיה אלגנטית, ומטרתו להציג שירות מקצועי וייחודי בתחום עיצוב האירועים.

## טכנולוגיות בשימוש
- HTML5  
- CSS3  
- Google Fonts (Heebo, Playfair Display)  
- Font Awesome Icons  
- Responsive Layout with CSS Grid

## תכונות עיקריות
- טופס יצירת קשר עם שדות נגישים  
- גלריית עבודות בצדדים  
- אייקונים מתחלפים ואינטראקטיביים  
- עיצוב מודרני ב־RTL  
- מותאם למסכים שונים
