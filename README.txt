
# Yomna Khashan | פרויקט סיכום - פורטפוליו אישי

ברוכים הבאים לפרויקט הסיכום שלי במסגרת קורס Full Stack Development.

האתר הזה נבנה כדי להציג את כישורי בתחום בניית אתרים, עיצוב ממשקים, פיתוח פרונט אנד, ורספונסיביות למובייל. האתר כולל דף בית, אזור אודות, גלריית פרויקטים, טופס יצירת קשר, ותשעה פרויקטים עם דפים פנימיים.

---

## מבנה האתר

- **index.html** - דף הבית הראשי עם וידאו רקע, תפריט ניווט, ואזור פתיחה.
- **projects/projectX/index.html** - תצוגת הפרויקט בדף סטטי.
- **projects/projectX/landing.html** - דף נחיתה מעוצב לכל פרויקט.

---

## טכנולוגיות בשימוש

- HTML5
- CSS3 (כולל Flexbox ו־Grid)
- Responsive Design עם Media Queries
- JavaScript בסיסי (לגלילה חלקה)
- Font Awesome Icons
- Google Fonts: Heebo, Playwrite AU SA

---

## התאמה לניידים

בוצעה רספונסיביות מלאה לכל הדפים באתר – כולל תפריט ניווט, טפסים, גלריה, ותצוגות הפרויקטים. ההתאמה כוללת שינויים בפריסת Flex, שינויי גודל לגופנים ולתמונות, ווידאו, ושדות טפסים.
